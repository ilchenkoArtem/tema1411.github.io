На второстепенных страницах у блока header должен быть класс .header__secondary-page
Например: <header class="header header__secondary-page"></header>

// краточка товара
Рейтинг(звезды)
Колличество закрашеных звёзд определяется классом stars-container-n, где n-колличество
закрашеных звёзд от 1-5
Например: <ul class="stars-container stars-container-4"> - закрашено 4 звезды

// каталог
Рейтинг(звёзды): аналогично с картой товара

Ссылки на страницы
Активная страница имеет класс: catalog__pages-link--active;
Неактивная стрелочка(переключатель) имеет класс: catalog__pages-button--disabled

//Корзина
Активная навигация шага(order__step) имеет класс (order__step--active);
Активный блок с шагом(step) имеет класс (order-visible)
